> 准备

```shell

// ssh
git clone git@github.com:SCWR/music-box.git

// https
git clone https://github.com/SCWR/music-box.git

yarn install

```

> 开发

```shell

// start react server
yarn run start

// start electron
yarn run electron

```

> 打包

```shell

yarn run package

```
